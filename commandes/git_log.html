Git log

  • Accueil

Git log vous donne la liste des commits qui ont été effectués dans le répertoire. Cela vous permettra de voir les modifications qui ont été faites. Cela vous donnera aussi le numéro de commit qui pourra vous servir pour d'autres commandes.

Syntaxe :

git log
commit 62cffc0f77d751afb08d87973dc8807e44c56ba7 Merge:9da4542 5fcc6c5 Author: Da**** T***** Date: Mon Mar 9 07:37:29 2015 +0100 Résolution conflit dans git-revert.html, et merge

Contrairement au git log -p, qui ne montre que les diffs entre les différents commits d'une branche, le git log --full-diff indique les différences entre les commits de la branche spécifiée et l'ensemble du projet, donc toute la différence, d'où son nom.

Syntaxe

git log --full-diff
user@user-VirtualBox:~/Bureau/Documents/git$ git log --full-diff commit 647145e1e748df6eb8b3a128293a95e8ec13d3e7 Author: user Date: Wed Feb 18 16:45:46 2015 +0100
