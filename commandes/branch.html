Git branch

Definition de branch

Description

Git branch est la commande qui permet de lister, créer ou detruire des branches. Sans argument, la commande liste simplement les branches existantes et met en valeur la branche courante avec une astérisque.

Syntaxe :

git branch
* master

Git branch --track

Description

Git branch --track (ou -t) permet, quand vous créez une nouvelle branche de marquer cette branche comme branche fille de la branche courante. Cette configuration permettra à git status et git branch -v de montrer la relation entre les branches.

Syntaxe :

git branch -- track
* master

branch --set_upstream

Description

La commande branch --set-upstream montre la branche sur laquelle on travail.

Syntaxe :

git branch --set-upstream
* master

branch_upstream-to/ -u

Description

On utilise la commande "branch --set-upstream-to=" ou "-u" pour creer une nouvelle branche master d'un fork creer à partir d'une branche master locale.

Syntaxe :

git branch --set-uptream-to=my_fork/master
Branch gh-pages set up to track remote branch master from my_fork.
