git clone --bare

  • Accueil

le git clone --bare crée un clone vide. Au lieu de créer un nouveau dossier dépot et de placer les fichiers administratifs dans le .git de ce dossier, le dossier lui-même devient le GIT_DIR. Cela implique nécessairement le git clone -n, puisqu'il n'y a aucun endroit pour checkout l'espace de travail.

Syntaxe :

git clone --bare
