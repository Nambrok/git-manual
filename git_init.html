le git init --bare

  • Accueil

Le git init --bare crée un dépot git nu.Si l'environnement GIT_DIR n'est pas configuré, le dépot se crée dans le dossier courant

Syntaxe :

git init --bare
git_init --bare
Initialized empty Git repository in C:/users/Ilian
